# TAXFORM
Website for tax.

## Project setup

**JSON_SERVER_API**
```
step I: install json server

npm i -g json-server

step II: install nodemodule

npm i
```
**VUE_WEBAPP_TAX**
```
step I: install nodemodule

npm i
```

### Compiles and hot-reloads for development
```
STEP I: Run JSON_SERVER_API

json-server datas.json

STEP II: run VUE_WEBAPP_TAX

npm run serve
```
